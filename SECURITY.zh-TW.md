# 安全政策

[English](SECURITY.md) | 繁體中文

## 支援版本

安全修正會套用到最新標記版本與預設分支。

## 回報安全弱點

請使用本版本庫的 GitHub 私下弱點回報功能。回報內容不可包含版本庫密碼、救援金鑰、私人網路位址、個人檔案路徑或真實備份資料。

請提供：

- 受影響的提交或版本；
- 使用 placeholder 路徑的最小重現方式；
- 預期行為與實際行為；
- 問題是否可能暴露明文、損壞版本庫，或讓驗證錯誤地回報成功。

## 安全邊界

本專案將系統管理員存取視為可信任邊界，不宣稱能防禦已經控制 Windows 系統管理員帳戶的攻擊者。詳情請閱讀[威脅模型](docs/THREAT-MODEL.zh-TW.md)。
